PasswordMemo 100 File ver.1

---------------------------------------------------------------------------------------------------------------------

このWindowsアプリはログインIDやパスワードなどを、PCに簡単に保管や移動をするなどのために作りました。
この書類のフォルダーをコピーするだけで簡単に保管は出来ますが、平文のままなどの防犯の精度は高くないので、
御自身のPCのみで使用がお勧めです。

AutoHotkey ver2.0.26で作成したので、ver 1.0の環境でRunやコンパイルをすると、使用できないかもしれないので、
AutoHotkey ver2.0.26で「PasswordMemo_100-File.ahk」をRunやコンパイルをしてください。

ahkというオープンソースのアプリですので、コンパイルされた裏で送信されているウイルスなアプリな危険はありません。
