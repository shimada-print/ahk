# AutoHotkey
AHK([AutoHotkey](https://ja.wikipedia.org/wiki/AutoHotkey))とはWindowsOS上で動く、スクリプト言語です。すぐに簡単にWindowsアプリが作れます。  

AutoHotkey が特に向いているのは、Windowsを自動操作する小〜中規模ツールです。かなり実務寄りです。AHKがC++で出来ているのと、Win32 API直叩をきなので速く、C++よりもコード数が少ないので快適です。  

インストールも数MBでインストールが出来て、インストーラーでインストールですので簡単です。操作は右クリックするとEditなどを選ぶと編集などが出来ます。
